# Сайт Upark

Одностраничный лендинг для компании Upark, занимающейся производством мебели и малых архитектурных форм.

## Структура проекта


```
uparkSite/
├── index.html
├── public/
│ ├── css/
│ │ └── style.css
│ └── js/
│ └── script.js
└── README.md
```


## Особенности

- Полностью адаптивный дизайн
- Современный интерфейс с анимациями
- Бегущая строка с логотипами партнеров
- Интерактивная форма обратной связи
- Оптимизированная производительность

## Технологии

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (Vanilla JS)
- AOS (Animate On Scroll)
